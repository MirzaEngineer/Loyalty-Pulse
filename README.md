# 📱 Loyalty-Pulse (Mobile App & SDK)

A comprehensive mobile loyalty solution designed in **Figma** and developed using **React Native**. This app provides a secure and intuitive experience for users to manage rewards, featuring a robust OTP authentication system and a distributable SDK.

---

## 🚀 Key Features

* **Secure OTP Login**: Integrated one-time password (OTP) verification for secure user access.
* **Figma-to-Code**: Pixel-perfect implementation of high-fidelity designs.
* **Custom SDK/APK**: Includes a pre-built SDK file for instant installation and testing.
* **API Integration**: Dynamic data handling via RESTful APIs for real-time loyalty tracking.
* **User-Centric UI**: Focused on smooth animations and easy navigation.
